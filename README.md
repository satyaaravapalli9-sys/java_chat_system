# Java_Projects

This is a basic chat system project where two user interfaces can chat simultaneously. This project was developed during my early Java programming journey to explore network communication and simultaneous response handling.

How to run:
1. Open in VS Code or other IDEs
2. Run java server.java
3. Run java client.java
